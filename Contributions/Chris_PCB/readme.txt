PCB Layout for PPMScope 1.20 by Krzysztof Passowicz

This layout is optimized for single sided board etching.

Contents
========

(1) 28PPMScope v1_2.fpc - FreePCB layout (FreePCB can be downloaded at www.freepcb.com)
(2) bottom_copper.pdf - PDF of bottom side of the layout
(3) bottom_copper.png - Image of bottom side of the layout
(4) bottom_copper.grb - Gerber of bottom side of the layout
